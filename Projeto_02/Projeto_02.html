• Introduo

Esta documento tem como objetivo apresentar a analise de dados de um arquivo com 891 registros disponibilizados pela Udacity. Esta analise no conclusiva e faz parte do meu estudo em data science. O arquivo contem informaes de alguns dos passageiros a bordo do navio Titanic. A analise ser feita no escopo de dados e quantidades do arquivo.

Com a analise deste aquivo pretendo responder algumas perguntas iniciais que surgiram ao visualizar o dicionrio de dados presente no kaggle:

1. Quantas passageiros morreram.
2. Existiu alguma relao de sobrevivncia com o sexo do passageiro.
3. A classe do passageiro influenciou na sobrevivncia?
4. Qual a faixa de idade mais presente?
5. A idade do passageiro influenciou na sobrevivncia?

## Limpeza de dados ##

# Substituindo dados para uma melhor leitura. Origem das informaes: https://www.kaggle.com/c/titanic/data

# Traduzindo o enum 0 e 1 para no e sim.
df['sobrevivente'] = df['sobrevivente'].replace([0, 1], ['no', 'sim']);

# Traduzindo o sexo do ings para o portugus.
df['sexo'] = df['sexo'].replace(['male', 'female'], ['homem', 'mulher']);

# Traduzindo as letras que representam o porto de embarque.
df['porto_embarque'] = df['porto_embarque'].replace(['C', 'Q', 'S'], ['Cherbourg', 'Queenstown', 'Southampton']);

# Tratando os valores nulos no numricos para facilitar a leitura de outras pessoas.
df["cabine"].fillna("no registrado", inplace = True);
df["porto_embarque"].fillna("no registrado", inplace = True);

# Preenchendo as idades nulas com a mdia da idade dos passageiros.
df['idade'] = df['idade'].fillna(df['idade'].mean());

qtd_total = len(df);
qtd_homens = len(df.query('sexo == "homem"'));
qtd_mulheres = len(df.query('sexo == "mulher"'));

fig, ax = plt.subplots(figsize=(10, 6), subplot_kw=dict(aspect="equal"))

data = [qtd_homens, qtd_mulheres]
labels = ['Homens', 'Mulheres']


def func(pct, allvals):
    absolute = int(pct/100.*np.sum(allvals))
    return "{:.1f}%\n({:d})".format(pct, absolute)

wedges, texts, autotexts = ax.pie(data, autopct=lambda pct: func(pct, data), textprops=dict(color="w"))

ax.legend(wedges, labels,
          title="Sexo",
          loc="center left",
          bbox_to_anchor=(1, 0, 0.5, 1)
         )

plt.setp(autotexts, size=15, weight="bold");

ax.set_title('Quantidade de passageiros por sexo' + ' (Total: ' + str(qtd_total) + ')');

  • Concluso

Apesar do arquivo no possuir os dados de todos os passageiros foi possivel levantar algumas concluses no definitivas do naufrgio. Percebi que existia uma diferena grande entre homens e mulheres, sendo 263 homens a mais do que mulheres, e que sobreviveram 124 mulheres a mais que homens. A maior parte dos passageiros estavam na terceira classe, principalmente os homens. Infelizmente a terceira classe teve o maior ndice de mortos, provavelmente pela localizao das hospedagens e priorizao nos botes. Identificado que existiu uma quantidade a maioria das mulheres tinham idade em torno dos 30 anos e que a maior parte dos falecidos homens tinha idade em torno de 20 a 30 anos.
